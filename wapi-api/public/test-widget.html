Chat Widget Test

This page loads the Botfeed chat widget for testing purposes. Look for the chat bubble in the bottom-right corner.

How it works

The widget script is loaded at the bottom of this page. It connects to your local API server at localhost:5001, fetches the widget configuration, and renders the chat bubble.

Testing steps

1. Make sure you have a flow with a Chatbot Widget trigger saved in the Flow Builder.
2. After saving, the trigger node will show an embed code with your widget API key.
3. Replace YOUR_WIDGET_API_KEY in this page's script tag with that key.
4. Refresh this page and click the chat bubble to test.

Note: Open your browser console (F12) to see widget logs. If the widget doesn't appear, check that the API key is correct and the server is running on port 5001.
